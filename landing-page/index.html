Perguntas frequentes

Tudo o que você precisa saber sobre o ToolSearch

Como funciona o sistema de QR Codes?

Cada ferramenta recebe um QR Code único que é gerado quando você cadastra o item no sistema. Os usuários podem escanear esse código com o aplicativo ToolSearch para registrar empréstimos, devoluções e acompanhar o histórico de uso.

É possível usar o ToolSearch sem conexão com a internet?

Sim, nosso aplicativo possui modo offline que permite registrar empréstimos e devoluções mesmo sem conexão. Os dados são sincronizados automaticamente quando a conexão é restabelecida.

Como o sistema registra o fluxo de ferramentas?

O ToolSearch registra cada etapa do ciclo de uso da ferramenta: quem pegou emprestado, quando, para qual finalidade e localização, e quando foi devolvida. Todos os dados ficam armazenados para consulta posterior e geração de relatórios.

Quanto tempo leva para implementar o sistema?

A implementação básica leva apenas alguns dias. Nossa equipe trabalha para que você consiga cadastrar todas as ferramentas e treinar a equipe em uma semana. Para personalizações mais avançadas, o prazo pode ser de 2 a 3 semanas.
